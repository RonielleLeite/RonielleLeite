**Tecnologias**: HTML5 • CSS3 • JavaScript • GitHub Pages

---

### 📖 [Manipulando DOM](https://github.com/RonielleLeite/Manipulando-DOM-js)
Projeto educacional focado em técnicas avançadas de manipulação do Document Object Model com JavaScript. Inclui exemplos práticos e bem documentados para aprendizado.

**Tecnologias**: JavaScript • DOM API • Git

---

**Tecnologias**: HTML5 • CSS3 • Bootstrap • GitHub Pages

---

Aplicação React que permite buscar e explorar perfis do GitHub em tempo real. Consumo da GitHub API com interface limpa e responsiva.

**Tecnologias**: React • JavaScript • GitHub API • CSS3
